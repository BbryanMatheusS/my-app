# Bem vindo ao meu Projeto 

## Para Instalar
- Crie um clone do projeto.
- Abra o projeto com VScode.
- Abra o CMD ou o Powershell na pasta pelo VScode.
- Digite o codigo "npm install" e aguarde o projeto instalar todas as dependencias.

## Para iniciar o projeto
- Abra a pasta do projeto com VScode.
- Abra o CMD ou o Powershell Na pasta . com ou sem o VScode.
- inicie o projeto com o comando "npm start" e aguarde o pojeto iniciar em uma aba de seu navegador.
